### 🔧 **방어구 모듈 제작**

받는 대미지 감소, 환경 적응, 스킬 3개의 모듈로 자신만의 방어구를 커스텀

<div align="center">
  <img src="/Images/armor1.gif" width="600" alt="모듈">
</div>

- **슬롯1: 받는 대미지 감소** - 업그레이드된 모듈일 수록 감소률은 높아짐
- **슬롯2: 환경 적응** - 춥거나 더운 지형에서의 저항
- **슬롯3: 스킬** - 이속, 공속, 투명화 등 스킬

**모듈 제작 특징:**
- 스토리가 진행될 수록 풀리는 모듈로 난이도 조정
- 자신만의 방어구를 만드는 것으로 재미 향상

## 🏆 목표 및 성취 시스템

### **📡 메인 스토리 진행**

| 진행 단계 | 달성 조건 | 스토리 변화 |
|-----------|-----------|-------------|
| **생존 적응** | 기본 생존 시설 구축 | 절망에서 희망으로 |
| **기술 복구** | 신호기 1차 수리 | 긴급 신호 송신 성공 |
| **통신 확장** | 신호 증폭 완료 | 궤도 위성과 접촉 |
| **지구 연결** | 최종 업그레이드 | 지구 본부 직접 통신 |
| **구조 대기** | 착륙장 확보 | 구조선 도착 준비 |

### **🏅 추가 성취 목표**
- 🔬 **완전한 외계 생물 도감**: 모든 생명체 연구 완료
- 🏗️ **최고 수준 기지**: 모든 시설 최대 업그레이드  
- ⚔️ **무기 마스터**: 전체 무기 제작 트리 완성
- 🌍 **행성 탐험가**: 모든 지역 및 숨겨진 장소 발견
- ⚡ **효율성 전문가**: 최소 자원으로 최대 효과 달성
